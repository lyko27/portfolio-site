# Mon Portfolio Personnel

Bienvenue dans la documentation de mon portfolio ! Ce site a été conçu pour être bien plus qu'un simple CV : c'est une véritable démonstration de mes compétences en développement et en conception logicielle.

## L'objectif
L'idée était de créer une vitrine web **ultra-rapide**, **fluide** et **épurée**, qui reflète mon profil d'élève ingénieur. Le design mélange une esthétique moderne avec des touches techniques (comme le terminal interactif ou les fonds "blueprint").

## Les technologies utilisées
Pour garantir des performances maximales, j'ai choisi une approche minimaliste :
- **Astro.js :** Un framework moderne qui génère des sites statiques ultra-légers.
- **Tailwind CSS :** Pour sculpter le design sur mesure directement dans le code.
- **Vanilla JavaScript :** Toutes les animations (terminal, menu flottant, filtrage des compétences) sont codées "à la main", sans utiliser de gros frameworks lourds.
