# My Personal Portfolio

Welcome to the documentation of my portfolio! This website was built to be more than just a resume: it's a practical demonstration of my software engineering and front-end development skills.

## The Goal
The main idea was to create an **ultra-fast**, **smooth**, and **clean** showcase that reflects my engineering background. The design blends a modern aesthetic with technical touches (like the interactive terminal and blueprint backgrounds).

## Tech Stack
To ensure maximum performance, I opted for a lightweight and minimalist approach:
- **Astro.js:** A modern framework that generates incredibly light static websites.
- **Tailwind CSS:** To craft a custom, responsive design directly in the markup.
- **Vanilla JavaScript:** All interactions and animations (terminal, smart header, skills toggle) are hand-coded from scratch, without relying on heavy frameworks.
